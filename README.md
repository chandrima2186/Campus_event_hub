Project Title: Campus Event Hub 

Overview:
Campus Event Hub is a web-based platform where students can browse, search, and register for university events.
It also provides an admin panel for managing events, approvals, and monitoring system activity efficiently.

Group Details:
Group Number: 06
Course Name: Database Management System
Instructor: Fahmidur Rahman Sakib 

Objectives of the System:

•Provide a digital platform formanaging university events.
•Allow students to easily discover and register for events.
•Enable administrators to manage events efficiently. 
•Maintain structured event data using a relational database. •Demonstrate full CRUD operations as required in the DBMS course. 

Features:

✅ User Registration & Login (with password hashing using bcrypt)
✅ Event Listing with Search (Debounce)
✅ Event Registration System
✅ My Events 
✅ Admin Panel (Add, Update, Delete Events)
✅ Approval / Reject System for registrations
✅ Logout System
